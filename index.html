📖 Как пользоваться

1. Загрузка данных

Загрузите файл с названиями и адресами. Нажмите кнопку ? внизу справа для подсказок.

2. Геокодирование

Нажмите "🚀 Начать геокодирование" для поиска координат по адресам.

3. Результаты

Сохраните результаты в Excel или рассчитайте маршрут.

4. Подсказки

Нажмите кнопку ? в правом нижнем углу для включения интерактивных подсказок. Наведите курсор на элемент для получения подсказки.
