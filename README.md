# GeoDude

Веб-приложение для геокодирования адресов и расчёта маршрутов.

## Возможности

- Загрузка адресов из файла (.txt, .xlsx, .csv)
- Автоматическая очистка и нормализация адресов
- Поиск координат через OSM (Nominatim, Photon)
- Расчёт расстояний от базовой точки
- Генерация матрицы расстояний
- Расчёт маршрутов
- Быстрый поиск объектов
- Экспорт результатов в Excel

## Формат входного файла

Название и адрес, разделённые Tab:

```
Название    Адрес
Объект 1   Область, Район, Город, Улица 1
Объект 2   Область2, Район2, Город2, Улица 2
```

## Быстрый старт

1. Откройте `index.html` в браузере
2. Загрузите файл с адресами
3. Нажмите "Начать геокодирование"
4. Сохраните результаты

## GitHub Pages

Приложение автоматически деплоится на GitHub Pages при пуше в main.
